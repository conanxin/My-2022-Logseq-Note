# Tools for Thought / 可编程笔记

## 主题简介

这条主线关注思考工具、可编程笔记、动态媒介和个人知识系统。笔记中的工具不是孤立产品，而是被放在 Engelbart、Ted Nelson、Alan Kay、Bret Victor、Dynamicland 等更长的计算媒介传统中理解。

## 2022 年笔记中的代表性线索

- Heptabase 被记录为一种面向卡片、白板和知识操作系统的工具尝试。
- Programmable Notes 将笔记从静态文本推进到规则、触发器和自动化动作。
- Subconscious 和反向链接线索体现了双链笔记的日常使用想象。
- Dynamicland 与 Ink & Switch 的动态模型把“直接操纵”和“看见系统”放在核心位置。
- Engelbart、Ted Nelson、Alan Kay、Bret Victor 形成了思考媒介的历史纵深。

## 关键标签

`#工具`、`#设计`、`Heptabase`、`Programmable Notes`、`Dynamicland`、`Engelbart`、`Alan Kay`、`Bret Victor`

## 可继续整理的问题

- 笔记系统什么时候从“保存信息”变成“帮助思考”？
- 可编程笔记应该对普通用户暴露多少程序能力？
- 白板、卡片、双链、时间线分别适合什么认知任务？
- Dynamicland 代表的是一种新交互范式，还是一个难以复制的研究环境？
- AI 进入笔记工具后，会增强思考，还是把思考外包给生成结果？

## 2026 年回看价值

这批材料可以帮助重新理解个人知识管理。它提醒我们，笔记工具的核心问题不是“哪个软件最好”，而是怎样让人更好地表示问题、建立连接、运行模型、进行长期思考。2026 年回看，AI 笔记工具已经很多，但这些早期线索仍然能提供判断标准。
