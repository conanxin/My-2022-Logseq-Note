# 读者入口

这份索引用来帮助读者从一个旧 Logseq graph 进入主题。原始笔记仍保留在 `journals/` 和 `pages/` 中；这里做的是重新分线、解释和指路。

## 1. Web3 / 区块链 / DAO

[进入主题页](docs/topics/web3-blockchain-dao.md)

2022 年 5 月，这条线索非常密集：以太坊、Web3 应用架构、DAO、DeSoc、NFT、稳定币崩盘和区块空间都出现在短短几天的笔记里。当时的关注点不是单纯追热点，而是在追问去中心化到底能否改变互联网、组织和所有权结构。2026 年回看，这组笔记的价值在于它记录了 Web3 高潮期的概念热度，也记录了 Luna/UST 崩盘等现实校正。它适合继续整理成一篇关于“早期 Web3 想象与治理困境”的回顾文章。

## 2. AI / 大语言模型 / 文明

[进入主题页](docs/topics/ai-and-civilization.md)

这条主线记录了 GPT-3、语言模型、常识推理、DeepMind 与 AlphaFold 等内容。2022 年的笔记已经在关心一个问题：AI 不是只会生成文本的工具，它会影响科学、认知、文明目标和社会制度。2026 年回看，这些材料很像大模型普及前夜的观察切片。它们适合继续整理成“AI 从能力展示走向文明问题”的阅读札记。

## 3. Tools for Thought / 可编程笔记

[进入主题页](docs/topics/tools-for-thought.md)

这条线索集中在 Heptabase、Programmable Notes、Subconscious、Dynamicland、Engelbart、Ted Nelson、Alan Kay 和 Bret Victor。2022 年我在关注的不是某个单一笔记软件，而是人如何借助计算媒介扩展思考、写作、协作和理解复杂系统的能力。2026 年回看，这些材料可以帮助解释为什么“个人知识管理”不只是文件夹整理，而是一种关于媒介、表示和认知工具的长期问题。

## 4. 元宇宙 / 游戏 / 虚拟世界

[进入主题页](docs/topics/metaverse-games-virtual-worlds.md)

这条主线把 Metaverse、Fortnite、VR、Will Wright、游戏开发者大会和虚拟空间设计放在一起。2022 年的关注点一方面是科技公司对元宇宙的叙事，另一方面是游戏作为复杂系统、教育媒介和社会空间的可能性。2026 年回看，这些笔记能帮助我们区分“元宇宙营销词”和“虚拟世界设计问题”。它们适合继续写成关于虚拟空间想象力边界的文章。

这条主线把 solarpunk、Low-Tech Magazine、Hundred Rabbits、Uxn、IPFS、旧技术和长期保存放在一起。2022 年我在寻找一种和永远在线、不断升级、订阅驱动相反的技术气质：更耐久、更节制、更离线、更能被人长期掌握。2026 年回看，这些笔记的价值不在于预测了某个产品，而在于保留了一种“技术也可以慢下来”的问题意识。
